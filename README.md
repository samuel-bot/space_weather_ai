🛰️ SpaceWeather AI — Global Solution 2026.1
Projeto desenvolvido para a disciplina de Cognitive Computing da FIAP.

📌 Sobre o Projeto
O desafio Space Connect nos propôs a integrar a economia espacial com problemas reais da Terra. O SpaceWeather AI é uma solução que utiliza inteligência artificial e dados atmosféricos — análogos aos coletados por satélites meteorológicos — para classificar o clima em tempo real (Sunny, Cloudy, Rainy, Snowy).

🚀 Problema
A grande quantidade de dados gerada por satélites meteorológicos (temperatura, pressão, umidade, UV) é subutilizada devido à complexidade da interpretação manual. O mercado precisa de sistemas capazes de processar esses fluxos de dados de forma automática e instantânea para tomadas de decisão rápidas no agronegócio, logística e segurança pública.

💡 Solução
Desenvolvemos um pipeline de Machine Learning que:

Ingere dados atmosféricos reais via API (Open-Meteo).

Processa e normaliza as variáveis críticas.

Classifica o clima com alta precisão utilizando modelos de ensemble learning.

Entrega insights acionáveis para qualquer coordenada geográfica, provando a eficácia da IA aplicada ao monitoramento espacial.

🛠️ Tecnologias e Bibliotecas
Linguagem: Python 3

Ambiente: Jupyter Notebook / Google Colab

Bibliotecas de Ciência de Dados:

pandas & numpy (Manipulação de dados)

matplotlib & seaborn (Visualização e EDA)

scikit-learn (Modelagem, pré-processamento e métricas)

APIs: Open-Meteo (Coleta de dados climáticos em tempo real)

📈 Resultados
O modelo foi validado através de técnicas de validação cruzada e obteve uma acurácia superior a 90%. A análise de Feature Importance confirmou a relevância científica das variáveis atmosféricas escolhidas no sucesso da classificação.
